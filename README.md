# Population_Power_BI_Analysis
#### This project involves combining and analyzing over 150 years of historical data to uncover meaningful trends and insights. The data was cleaned, filtered, and transformed using a Star Schema data model to improve data organization and analytical performance.

#### Power BI, and data modeling techniques, the project focused on building efficient relationships between fact and dimension tables, enabling accurate reporting and interactive dashboard analysis. The analysis helped identify patterns, trends, and key metrics from large-scale historical datasets through effective visualization and business intelligence practices.
